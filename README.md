# KOReader Toolbox

Personal repo for KOReader-related plugins and patches.

## What is here

- `plugins/` contains a modified version of the Readeck plugin originally by [Ice Year](https://github.com/iceyear/readeck.koplugin).
- `patches/` contains the patch files I apply to KOReader.
- `patches/` also includes two patches that fix the progress bar (the file must be opened once to initialize, Readeck trophies when complete).

## Notes

- This repo is a staging area for my KOReader tweaks.
- Expect local changes and experiments in-place.
